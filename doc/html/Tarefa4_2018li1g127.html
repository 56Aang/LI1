auxalteramapaCH :: Mapa -> Posicao -> Int -> Mapa #

recebe o mapa, a posição onde os blocos se localizam, e um inteiro, que vai servir para destruir os 2 blocos que a bala tem contacto | se o bloco for destrutivel, então substitui o bloco por um bloco vazio e repete o processo para o bloco a baixo | se o bloco for indestrutivel, apenas repete o processo para o bloco a baixo

auxalteramapaCV :: Mapa -> Posicao -> Int -> Mapa #

recebe o mapa, a posição onde os blocos se localizam, e um inteiro, que vai servir para destruir os 2 blocos que a bala tem contacto | se o bloco for destrutivel, então substitui o bloco por um bloco vazio e repete o processo para o bloco ao lado | se o bloco for indestrutivel, apenas repete o processo para o bloco ao lado

Separa lista de disparos que chocam só contra uma parede

mataj :: [Jogador] -> [Disparo] -> [Jogador] #

recebe uma lista de jogadores e uma lista de disparos | subtrai todas as vidas dos jogadores que intersetam disparos com a auxiliar subtraivida

subtraivida :: Jogador -> [Disparo] -> Jogador #

Recebe um jogador e uma lista de disparos; se o jogador interceta com algum disparo, diminui 1 vida

mantemdisp :: [Disparo] -> [Disparo] #

recebe a lista de disparos iniciais e mantêm apenas os que não são canhão

separacanhao :: [Disparo] -> [Disparo] #

Fica com uma lista só com disparos de canhão

modificadisparo :: Mapa -> [Jogador] -> [Disparo] -> [Disparo] #

recebendo um mapa e uma lista de disparos, altera a lista de disparos, avançando aqueles em que a sua posição é vazia e retirando aqueles em que não é (choca com alguma coisa)

elimina :: [Disparo] -> [Disparo] #

recebe uma lista de disparos e remove todos os disparos em que a posição é a mesma | se a lista for vazia, devolve a mesma, caso contrário, verifica se o primeiro disparo tem algum outro com a mesma posição; se sim, então chama uma auxiliar para eliminar todos os disparos com essa posição; caso contrário, mantêm o disparo e verifica o mesmo para todos os disparos seguintes

elemdisparos :: Disparo -> [Disparo] -> Bool #

recebe um disparo e uma lista de disparos | se a posição do disparo for igual ao do primeiro da lista, então retorna Verdade; caso contrário, verifica para o resto da lista

eliminarepetidas :: Disparo -> [Disparo] -> [Disparo] #

elimina todos os disparos com a mesma posição do disparo dado | se a posição do primeiro disparo for igual ao do primeiro da lista, então elimina o primeiro; caso contrário, verifica-o para o resto da lista

avancabala :: Disparo -> Disparo #

recebendo um disparocanhao, avança esse disparo consoante a direção dada | Dependendo da direção, move a bala de canhão um bloco

verificadisparocostas :: Disparo -> [Disparo] -> Bool #

recebe um disparo e uma lista de disparos e verifica se existe algum disparo nessa lista que esteja virado de costas para o mesmo

eliminacostas :: [Disparo] -> [Disparo] #

Recebe uma lista de disparos e, usando uma auxiliar que elimina, numa lista de disparos, os disparos que estão nas costas, elimina todos os disparos numa lista que se encontram nas costas

auxeliminacostas :: Disparo -> [Disparo] -> [Disparo] #

recebe um disparo e uma lista de disparos | se o disparo tem direção oposta, e está nas costas de outro disparo, ambos os disparos são eliminados
